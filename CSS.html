Introduction

CSS stands for Cascading Style Sheets, and it is used to control the look and feel of web pages. With CSS, you can modify the visual presentation of HTML elements, such as setting text color, size, font, margins, padding, positions, backgrounds, and more.

There are three main ways to apply CSS to HTML:

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS

Inline CSS is applied directly to HTML elements using the style attribute.

It is generally used for quick styling changes or when you need to apply a unique style to a single element.

Syntax:

<element style="...style rules...">Content</element>

Example:

<h1 style="color: red;">This is a red heading</h1>

Note: It's not recommended to use inline CSS for large projects because it can make code harder to manage.

Internal CSS

Internal CSS is written within a single HTML document, inside the <style> tag in the <head> section. It is useful when styling needs to be applied to one specific page.

Syntax:

<style>
selector {
  property: value;
  property: value;
}
</style>

External CSS

External CSS is stored in an external file, which is linked to the HTML document using the <link> tag. This method is ideal for large projects where you want to apply the same styles across multiple pages.

Syntax:

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

CSS Selectors

CSS selectors are used to target HTML elements that you want to style. There are various types of selectors:

  • Element Selector - targets all elements of a specific type (e.g., p, h1).
  • ID Selector - targets a single element with a specific ID using the # symbol (e.g., #myDiv).
  • Class Selector - targets elements with a specific class using the . symbol (e.g., .myClass).
  • Attribute Selector - targets elements with specific attributes (e.g., [type="text"]).

CSS Box Model

The CSS Box Model is a fundamental concept that defines how elements are structured in CSS. It consists of:

  • Content - The content of the element (e.g., text, image).
  • Padding - The space between the content and the border.
  • Border - The edge around the padding and content.
  • Margin - The space outside the border.

CSS Flexbox

CSS Flexbox is a layout model that provides an efficient way to arrange items in a container. It allows items to align and distribute space dynamically within a flex container.

  • display: flex; - Defines a flex container.
  • justify-content - Aligns flex items along the main axis (e.g., justify-content: center;).
  • align-items - Aligns flex items along the cross axis (e.g., align-items: stretch;).

CSS Grid

CSS Grid is another layout model that provides a two-dimensional grid-based layout system. It is ideal for building complex, responsive layouts.

  • display: grid; - Defines a grid container.
  • grid-template-columns - Specifies the columns in the grid (e.g., grid-template-columns: repeat(3, 1fr);).
  • grid-gap - Defines the spacing between grid items (e.g., grid-gap: 10px;).
