Introduction
CSS stand for cascading Style sheet which is used to format the layout of the web page. With CSS, you can control the color, size of the text, font, margin between the elements, padding, position, background image or colors and different display for different devices, screen size and much more. CSS can be applied to HTML file in three different ways:
1) Inline
2) Internal CSS
3) External CSS
• Inline CSS
Inline CSS is applied to the HTML element within the same line by using style attribute.
Syntax:
<'element style = “...style rules....”>…………...<'/element style>
Example:
<'h1= “color: red;” >This is red heading <'/h1>
•Internal CSS
Internal CSS is applied to the html document by using a <'style> element with in <'head> section. Internal CSS is use to define a style for single HTML page.
Syntax:
<'style> Tag{attribute:value;attribute:value;….} ………. <'/style>
